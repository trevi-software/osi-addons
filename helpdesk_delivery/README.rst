===================
Helpdesk - Delivery
===================

This module allows users to specify a delivery carrier on Helpdesk Tickets

Usage
=====

* Go to Helpdesk
* Create or select a ticket
* On the Inventory Tab select a carrier

To close a ticket, all the related service orders must be closed.
